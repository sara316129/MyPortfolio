Project 01

Nike Product Advertisement

Two scenes - a shoe ad and a cap ad - assembled into a single composition with shared circle elements and brand typography.

Composition Structure

Each product scene is a separate pre composition nested inside a master "Final" composition. This keeps the timeline clean and allows independent editing of each scene without affecting the overall render.

Key Techniques

  • Kinetic typography with Range Selector Offset animation for text reveal
  • Floating blurred circle elements animated with Position keyframes for depth
  • Product PNG layers with Position + Rotation keyframed entry
  • Nike logo placed as a static branding layer above all content
  • Adjustment layer for global colour correction across scenes
Duration10 sec per scene
Frame Rate29.97 fps
Resolution1920 × 1080
Project 02

Neon Circuit Lines

A glowing circuit board animation using the Saber effect - multi coloured neon paths draw themselves across a dark background.

Saber Plugin Setup

Each glowing path is a separate shape layer with the Saber effect applied in Add blend mode. The Start Offset and End Offset properties are keyframed so each line draws on progressively, simulating circuit board activation.

Key Techniques

  • Shape layers drawn as angular paths to mimic PCB trace geometry
  • Saber effect in Add blend mode for natural glow compositing on dark backgrounds
  • Node circles animated with Position keyframes - appearing as connection points light up
  • Multiple Saber layers staggered in time to create a sequential draw-on effect
  • Colour variation across layers - cyan, magenta, orange, green - for multi channel visual
Duration10 sec
Frame Rate29.97 fps
Resolution1920 × 1080
Project 03

Countdown Timer

An animated countdown with dual colour glowing rings and bold number reveals - built using Saber's text layer core type.

Saber Core Type - Text Layer

Instead of a drawn path, Saber's Core Type is set to "Text Layer" - the glow follows the outline of a number character. As the number changes each second, the ring reshapes itself around the new digit automatically.

Key Techniques

  • Mask Evolution keyframed to rotate the glow ring continuously around each number
  • Two Saber layers stacked - one orange, one blue - offset by 180° for dual-colour ring effect
  • Start Offset and End Offset held at 60% to create an arc rather than a full circle
  • Glow Circle pre-composition with a Black Solid beneath for clean compositing
  • Glow Intensity 48%, Glow Spread 0.38, Core Size 3.00 for soft broadcast quality glow
Duration1 sec per digit
Frame Rate29.97 fps
Glow ColoursBlue + Orange
Project 04

Food Menu Advertisement

A multi scene animated burger menu ad with 3D card rotations, bold price reveals, and animated background shapes - designed for social media or restaurant display.

Multi Scene Structure

The composition contains four separate menu scenes (Scene 1-4), each featuring a different burger product. Each scene is a pre composition that enters and exits via Y Rotation and Z Rotation keyframes on the Classic 3D renderer - giving the card flip transition effect.

Key Techniques

  • Classic 3D renderer enabled for Y axis rotation card-flip transitions between scenes
  • Drop Shadow effect on each scene pre comp for depth and separation from background
  • Polystar shape layers in the background - rotated continuously as decorative elements
  • Product PNG layers animated with Position keyframes to slide in from off screen
  • Text layers for Item Name, Description, and Price each animated separately with staggered entry
  • Bold contrasting colour blocks - teal/red, orange/green - for visual hierarchy per scene
Duration40 sec total
Frame Rate29.97 fps
Scenes4 products
Project 05

Cinematic Story Sequence

A dramatic black and white cinematic piece - three spotlight shafts, a lone silhouette, film grain, and high contrast curves create an atmospheric narrative opening.

Lighting Setup

Three White Solid layers in Screen blend mode form the spotlight shafts. Each solid is masked with a tall tapered shape. The CC Spotlight effect is keyframed on the adjustment layer - animating the From and To coordinates to sweep the light beam across the scene over time.

Key Techniques

  • White Solid layers in Screen blend mode for additive light blending over a black background
  • CC Spotlight effect keyframed on the adjustment layer - From (911.6, 436.5) To (934.5, 487.8)
  • Film grain noise via the Noise effect (Amount 20%) on a separate adjustment layer for a classic cinematic texture
  • Curves effect on the adjustment layer - steepened S curve for high contrast black and white toning
  • Silhouette figure layer ("men") composited in Normal mode at the base of the light columns
  • Cinema 4D renderer enabled for depth-aware compositing

Technical Notes

Tools & Techniques

Adobe After Effects 2020

Primary compositing and animation tool. Used for keyframe animation, pre-composition workflow, blend modes, masks, and the Classic 3D and Cinema 4D renderers.

Saber Plugin

Third-party After Effects plugin used for glowing neon stroke effects. Applied in the circuit lines, countdown timer, and wherever high quality glow paths were needed. Supports Core Type Text Layer for shape following glows.

CC Spotlight & Noise

Built-in After Effects effects used in the cinematic story sequence. CC Spotlight creates the animated light beam sweep, while the Noise effect adds analogue film grain texture.

Keyframe Animation

All motion is driven by hand-crafted keyframes - Position, Rotation, Scale, Opacity, and effect specific properties. Range Selector Offset for text reveals, Mask Evolution for rotating glow rings, and Start/End Offset for Saber draw on animations.

Pre-Composition Workflow

Complex projects use a layered pre-comp structure - individual scenes or elements are pre composed and nested into a master timeline. This keeps projects manageable and allows 3D layer ordering and blend mode isolation per group.

Classic 3D & Cinema 4D Renderers

Classic 3D renderer used for card flip transitions in the food ad via Y/Z Rotation. Cinema 4D renderer enabled in the cinematic story for depth-aware layer ordering and advanced compositing.
