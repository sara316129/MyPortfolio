Sales Data Dashboard

Sales Data Analysis Project

Exploring Sales, Customers & Profitability with SQL & Power BI

This project uses a publicly available dataset from Kaggle, focusing on real-world retail sales data.

Data Cleaning & Preparation with SQL

The raw dataset was processed using SQL Server Management Studio (SSMS):

  • Duplicates were removed
  • Nulls handled appropriately
  • Consistent data types ensured
  • Returns matched to orders via Order ID

Interactive Dashboard in Power BI

After cleaning, data was loaded into Power BI to create powerful visualizations:

  • DAX used for KPIs: sales, profit, quantity
  • Interactive filters: year, segment, region
  • Custom date slicers & trend indicators
  • Drill-through for customer-level analysis

Customer Insights & Challenges

This project uncovered deep insights:

  • Low-margin/high-volume products identified
  • Multi-location customers handled in summaries
  • Year-over-year targets built with DAX logic
  • Visually balanced dashboard for stakeholder use
