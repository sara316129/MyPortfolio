Sales Data Analysis Project

Uncover insights into customer behavior, product performance, and profitability.

Data Cleaning & Preparation with SQL

The data was imported into SQL Server for cleaning and initial processing. Key steps included:

  • Removed duplicate records
  • Filtered and filled null values
  • Ensured consistent data types and formats
  • Matched returns to orders using `Order ID`

Interactive Sales Dashboard in Power BI

The cleaned data was loaded into Power BI to create an interactive and dynamic sales dashboard. Key features included:

  • KPIs for total sales, profit, and quantity
  • DAX formulas for profit margins and unit prices
  • Interactive slicers for customer segments and regions
  • Drill-through functionality to explore deeper customer-level data

Customer-Level Insights & Challenges

The dashboard helped uncover several key insights related to customer behavior, such as:

  • Identifying products with high sales but negative profits, which led to the discovery of low-margin or loss-leading products
  • Managing customers with multiple shipping addresses, ensuring accurate order summaries
  • Creating a balanced dashboard layout despite limited space, ensuring clarity and usability
